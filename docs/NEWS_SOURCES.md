## 5. 뉴스 vs 시장반응 괴리 감지 방법

### 로직
```
1. 뉴스 감성 점수 산출 (LLM 또는 사전 기반)
   - 매우 부정(-2) ~ 매우 긍정(+2)

2. 같은 날 시장 반응 측정
   - 종가 변화율 (%)

3. 괴리 판별:
   - 뉴스 감성 < -1 AND 시장 > +1% → "악재 무시 상승" ⚠️
   - 뉴스 감성 > +1 AND 시장 < -1% → "호재 무시 하락" ⚠️

4. 역사적 통계:
   - "악재 무시 상승" 후 30일 평균 수익률: +X%
   - "호재 무시 하락" 후 30일 평균 수익률: -X%
```

### 이것이 중요한 이유
> "끔찍한 뉴스가 나왔는데도 시장이 상승한다면 생각해봐야 할 것들이 생긴다.
> 예상을 빗나간 시장반응은 중요한 것을 암시하는 수가 있다." — 사용자 원칙

## 6. 구현 우선순위

| 순위 | 모듈 | 소스 | 난이도 | 이유 |
|------|------|------|--------|------|
| **1** | M2 매크로 | FRED API + BOK ECOS | 쉬움 | 무료 API, 정형 데이터, 즉시 가능 |
| **2** | M1 정량 심리 | VIX + Fear&Greed + 신용잔고 | 쉬움 | 이미 수치화된 지표 |
| **3** | M1 뉴스 감성 | RSS 수집 + LLM 감성 분석 | 중간 | RSS 무료, LLM API 비용 발생 |
| **4** | M1 커뮤니티 | 네이버·DC·에펨코리아 크롤링 | 중간 | 크롤링 유지보수 필요 |
| **5** | M1 괴리 감지 | 뉴스 감성 + 시장 데이터 결합 | 어려움 | 3번 완료 후 가능 |
