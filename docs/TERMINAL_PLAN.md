> 3대 축: **위험관리** · **최적 자산/타이밍** · **시장 심리**

## 1. 현재 상태 vs 목표 Gap 분석

### 축 1: 위험관리 (Risk Management)

| 기능 | 현재 | 목표 | 우선순위 |
|------|------|------|----------|
| 거래당 손실률 설정 (기본 2%) | ❌ 코드에 1% 고정 | 슬라이더로 조절 가능, 변경 즉시 반영 | **P0** |
| 손실률↔필요수익률 대응표 | ❌ | 손실 2% 설정 시 "원금 회복에 필요한 수익률: 2.04%" 즉시 표시 | **P0** |
| 포지션별 리스크 히트맵 | ❌ | 각 포지션이 Stop까지 몇 % 남았는지 시각화 | P1 |
| 드로다운 추적기 | ❌ | 현재 고점 대비 하락률, 역대 최대 드로다운 | P1 |
| 포트 전체 리스크 | ❌ | 모든 포지션 동시 손절 시 총 손실 금액/% | **P0** |
| 수정된 자본 기준 | ❌ | 손실 후 남은 자본 기준으로 자동 재산정 | P1 |
| 최대 동시 포지션 제한 | ❌ | 4~6개 넘으면 경고 | P2 |

**손실률↔필요수익률 계산식:**
```
필요수익률 = 1 / (1 - 손실률) - 1

예시:
  2% 손실 → 2.04% 필요  (거의 대칭)
  5% 손실 → 5.26% 필요
  10% 손실 → 11.11% 필요
  20% 손실 → 25.00% 필요  (비대칭 시작)
  50% 손실 → 100.00% 필요 (치명적)
```

### 축 2: 최적 자산/타이밍/손절/애드업

| 기능 | 현재 | 목표 | 우선순위 |
|------|------|------|----------|
| L0 글로벌 RS 랭킹 | ✅ | 유지 + 월간 추이 차트 | P1 |
| L1 섹터 RS | ⚠️ 부분 | 테마 ETF 20~30개 전용 스캔 | P1 |
| L2 종목 돌파 신호 | ✅ | 유지 | - |
| 진입 계산기 | ❌ | 종목 선택 → 자동으로 매수가·수량·손절가·목표가 계산 | **P0** |
| 애드업(피라미딩) 계산기 | ❌ | 현재 보유 기준 → 1차/2차 추가매수 가격·수량·새 Stop | **P0** |
| Trailing Stop 자동 상향 | ✅ | 유지 + 상향 이력 로그 | P2 |
| Time Stop 알림 | ❌ | 진입 후 2주 경과 + ±2% 미만 → 경고 | P1 |
| 돌파 품질 점수 | ❌ | 거래량 배수 + 수축 깊이 + RS 순위 종합 | P1 |
| 섹터 히트맵 | ❌ | 전 섹터 RS를 격자로 시각화 (강세=밝은, 약세=어두운) | P2 |
| 알림 시스템 | ❌ | 돌파 발생·Stop 이탈 시 알림 (이메일/텔레그램) | P2 |

**진입 계산기 예시 UI:**
```
┌─────────────────────────────────────────┐
│  진입 계산기                              │
│                                          │
│  종목: [KODEX반도체 ▼]                    │

│                                          │

│                                          │
│  ── 목표가 (R배수) ──                     │
```

## 3. 구현 로드맵

### Phase 1 (1~2주): 핵심 기능 (P0)
- [ ] 리스크 슬라이더 + 손실↔필요수익률 즉시 표시
- [ ] 포트 전체 리스크 (모든 포지션 동시 손절 시 총 손실)
- [ ] 진입 계산기 (종목 선택 → 수량·손절·목표 자동)
- [ ] 애드업 계산기 (1차·2차 추가매수 가격·수량·새 Stop)

### Phase 2 (2~3주): 분석 강화 (P1)
- [ ] VIX 역사적 백분위
- [ ] Fear & Greed 종합 지수
- [ ] 뉴스 vs 시장반응 괴리 감지
- [ ] Time Stop 알림
- [ ] 섹터 ETF 전용 RS 스캔
- [ ] FOMC FedWatch 확률

### Phase 3 (3~4주): 시각화·알림 (P2)
- [ ] 섹터 히트맵 (격자 시각화)
- [ ] 포지션 리스크 히트맵
- [ ] 비경제 커뮤니티 크롤러
- [ ] 알림 시스템 (텔레그램/이메일)
- [ ] 드로다운 추적기 + 이력
